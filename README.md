# Interesting Datasets

A collection of interesting datasets spanning food delivery, shipping, restaurants, judicial records, and more. Analyzed and visualized using [MinusX](https://minusx.ai).

## Datasets

| Dataset | Description | Source | Format | Folder |
|---------|-------------|--------|--------|--------|
| [adventureworks](./adventureworks/) | Classic AdventureWorks sample database (sales, products, customers, HR) | Microsoft sample database | DuckDB | `adventureworks/` |
| [mxfood](./mxfood/) | Synthetic food delivery business dataset (~35 tables, ~10M rows) covering orders, marketing, finance, B2B catering, ops, and consumer insights across SF & South Bay | Synthetically generated | DuckDB + CSV | `mxfood/` |
| [sfrestaurants](./sfrestaurants/) | Top-ranked Bay Area restaurants with cuisine, price, location, and reviews from the SF Chronicle | SF Chronicle | JSON + CSV | `sfrestaurants/` |
| [sfjudicial](./sfjudicial/) | San Francisco Superior Court data including cases, attorneys, judicial assignments, DA arrests/prosecutions, and charge dispositions | SF Open Data | Parquet | `sfjudicial/` |
| [strait_of_hormuz](./strait_of_hormuz/) | Daily ship transit data through the Strait of Hormuz (2019–2026) — vessel counts and cargo capacity by type | ArcGIS / EIA Chokepoints | CSV | `strait_of_hormuz/` |
